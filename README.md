# mail2json
Email to JSON/YAML converter for evaluating mail-parser lib

mail2json 0.1.0

USAGE:
    mail2json [OPTIONS] --input <input>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -i, --input <input>      
    -o, --output <output>
